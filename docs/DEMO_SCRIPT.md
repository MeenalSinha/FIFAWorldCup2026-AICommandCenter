# Judge Demo Script (5 minutes)

1. **Open on the Overview dashboard.** Point out this is one screen
   covering fans, operations, accessibility, sustainability, and
   transportation -- not eight disconnected tools.
2. **Stadium Digital Twin.** Show Gate C flagged high-density; read the
   AI Insight aloud: "Gate C will exceed safe capacity in 18 minutes."
   Emphasize this is a prediction, not a current-state alert.
3. **Ask the AI Assistant** a fan question ("Where can I find an
   accessible restroom near Gate B?") to show the same Gemini-backed
   reasoning powering the chat surface.
4. **Switch to Accessibility.** Show the wheelchair-route reroute
   triggered by the Gate B blockage insight -- accessibility isn't an
   afterthought, it's wired into the same event stream.
5. **Switch to Volunteers.** Show the AI-prioritized task list putting
   the Gate C team first, and file a sample incident to show the
   generated action-plan/SOP output.
6. **Switch to Reports.** Show the executive daily summary narrative
   tying the Gate C event, the sustainability score, and volunteer SLA
   performance into one paragraph an organizer can read in ten seconds.
7. **Close on Sustainability.** End on the green score ticking up as the
   "wow" beat: the platform isn't just about safety, it also proves
   measurable environmental impact match over match.
